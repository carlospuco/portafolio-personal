## Overview

Actuaria Plus is a web platform developed at Actuaria to support extended business and actuarial operations. Built on top of the same full-stack foundation as ActuaFast, it extends functionality to cover additional business processes and user workflows.

## Key Features

- **Full-Stack Development**: End-to-end development with a Python backend and React frontend, covering business processes specific to Actuaria's expanded operations.
- **RESTful APIs**: Designed and maintained REST APIs for business logic, data access and integration with internal services.
- **Cloud Infrastructure**: Deployed on Google Cloud Platform with automated CI/CD pipelines for continuous delivery and reliability.
- **Database Management**: Managed MySQL relational data and Firebase non-relational storage to cover different data access patterns.
- **Unit Testing**: Wrote unit tests to ensure code quality and prevent regressions across releases.

## Technologies Used

- **Python**: Backend REST API and business logic.
- **React / TypeScript**: Frontend SPA for end users.
- **Google Cloud Platform**: Hosting and cloud services.
- **MySQL & Firebase**: Data storage.
- **Docker & CI/CD**: Automated build and deployment.
- **Git**: Version control.

## Outcome

Actuaria Plus extended the company's digital capabilities, providing users with a reliable platform for managing business processes. Continuous deployment practices ensured fast iteration and stable production releases.
