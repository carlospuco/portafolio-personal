## Overview

Contributed to the development of [Banco Pichincha's Banca Web](https://www.pichincha.com/banca-web), one of Ecuador's largest online banking platforms serving individual customers, businesses, SMEs and microentrepreneurs. The platform enables 24/7 banking operations from any device — transfers, payments, account management and more.

## Key Features

- **Component Development**: Built reusable UI components with React following the internal C3 Design System, ensuring visual consistency across the platform's modules including account dashboards, transfer flows and payment screens.
- **Backend Integration**: Integrated frontend components with Java Spring Boot microservices via REST APIs to display real-time account balances, transaction history and card information.
- **Security-Sensitive UI**: Implemented UI flows respecting the platform's two-factor authentication (Clave Digital), account blocking logic and secure form handling.
- **UX/UI Standards**: Applied UX/UI best practices aligned with the bank's brand guidelines — tabbed navigation, quick-access icons, product carousels and responsive layouts for all devices.
- **Code Quality & Refactoring**: Refactored legacy components improving performance, readability and test coverage.
- **Agile Scrum**: Worked in sprint-based cycles with cross-functional teams including designers, backend engineers and QA.

## Technologies Used

- **React**: Component-based frontend development.
- **Java Spring Boot**: Backend microservices and REST API consumption.
- **C3 Design System**: Banco Pichincha's internal design system for consistent UI.
- **Git**: Version control and code review in a team environment.
- **Agile / Scrum**: Sprint planning, daily standups and retrospectives.

## Challenges and Learnings

Working in a large financial institution meant strict code review processes and high standards for security, accessibility and performance. Navigating the C3 Design System required learning its conventions quickly and contributing components that felt native to the system. Collaborating across frontend and backend disciplines strengthened full-stack thinking in a regulated, high-availability environment.

## Outcome

Delivered production-ready components integrated into Banco Pichincha's live banking platform, which serves thousands of users daily. Improvements in code quality and test coverage contributed to more stable releases and a faster development workflow across the team.
