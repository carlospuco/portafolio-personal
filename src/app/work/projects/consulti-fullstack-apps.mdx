## Overview

Developed an internal web platform for [Telconet](https://www.telconet.net/), one of Ecuador's leading telecommunications and technology companies with over 30 years of experience serving corporate clients across Latin America. The project was delivered through ConsulTI and covered full-stack development from API design to cloud deployment.

## Key Features

- **Full-Stack Development**: End-to-end development with a Python backend exposing RESTful APIs and a React frontend, covering the internal business processes specific to Telconet's operations.
- **API Design**: Designed REST APIs following best practices — consistent naming, proper HTTP semantics, error handling and versioning to support the frontend and future integrations.
- **Database Management**: Modeled and managed relational data in MySQL for structured business records and Firebase for real-time data needs.
- **Cloud Deployment**: Deployed the platform on Google Cloud Platform, managing environments and ensuring uptime for internal users.
- **CI/CD Automation**: Built automated pipelines for testing and deployment, reducing release friction and catching regressions early.

## Technologies Used

- **Python**: REST API development and backend business logic.
- **React / TypeScript**: Frontend SPA for internal users.
- **MySQL & Firebase**: Relational and real-time data storage.
- **Google Cloud Platform**: Hosting and cloud infrastructure.
- **Docker & CI/CD**: Containerization and automated delivery.
- **Git**: Version control and team collaboration.

## Challenges and Learnings

Building an internal platform for a telecommunications company meant understanding domain-specific workflows around connectivity, cloud services and enterprise operations. Working within ConsulTI as a contractor required fast onboarding into Telconet's processes and delivering production-ready features under tight timelines.

## Outcome

Successfully delivered and maintained the internal platform supporting Telconet's operational workflows. The cloud-hosted solution improved team productivity by centralizing processes that were previously handled manually or across disconnected tools.
