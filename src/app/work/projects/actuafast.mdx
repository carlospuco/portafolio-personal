## Overview

ActuaFast is a web platform developed at Actuaria to streamline actuarial and business processes. The system centralizes data management, automates calculations and exposes business logic through RESTful APIs consumed by a React frontend.

## Key Features

- **RESTful API Design**: Designed and implemented REST APIs in Python to handle actuarial calculations and business workflows, ensuring clean separation between frontend and backend.
- **React Frontend**: Built a responsive, component-driven UI in React with TypeScript, consuming backend APIs and delivering a smooth user experience.
- **Cloud Deployment**: Deployed and maintained the application on Google Cloud Platform (GCP), ensuring availability, scalability and reliability.
- **CI/CD Pipelines**: Implemented automated pipelines for build, testing and deployment, reducing manual intervention and accelerating release cycles.
- **Database Integration**: Worked with MySQL for relational data and Firebase for real-time NoSQL storage depending on use case requirements.

## Technologies Used

- **Python**: Backend development and REST API implementation.
- **React / TypeScript**: Frontend SPA development.
- **Google Cloud Platform**: Hosting, cloud services and infrastructure.
- **MySQL & Firebase**: Relational and non-relational data storage.
- **Docker & CI/CD**: Containerization and automated deployment pipelines.
- **Git**: Version control and collaborative development.

## Outcome

The platform streamlined actuarial workflows, reduced manual processing time and provided a reliable, cloud-hosted system accessible to the entire team. Automated pipelines improved deployment frequency and code quality through continuous integration.
