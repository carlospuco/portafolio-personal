## Overview

Developed an internal complaints and claims management portal for [Corporación Favorita](https://www.corporacionfavorita.com), one of Ecuador's largest retail corporations operating multiple supermarket brands nationwide. The project was delivered through ConsulTI and provided a centralized platform to register, track and resolve complaints across the organization.

## Key Features

- **Complaints Management**: Built a full workflow for complaint registration, assignment and resolution tracking, allowing internal teams to manage cases from submission to closure.
- **RESTful API**: Designed and implemented a Python backend exposing REST APIs to handle complaint data, user roles and status updates consumed by the React frontend.
- **Role-Based Access**: Implemented different access levels for employees, supervisors and administrators, ensuring each user sees and manages only what corresponds to their role.
- **Real-Time Updates**: Integrated Firebase for real-time status notifications, keeping users informed of complaint progress without manual page refreshes.
- **Database Design**: Modeled complaint records, categories, priorities and resolution history in MySQL for structured reporting and audit trails.
- **Cloud Deployment**: Deployed on Google Cloud Platform with CI/CD pipelines for automated testing and delivery.

## Technologies Used

- **Python**: Backend REST API and business logic.
- **React / TypeScript**: Frontend SPA for internal users across departments.
- **MySQL**: Relational data storage for complaints, users and audit records.
- **Firebase**: Real-time notifications and status updates.
- **Google Cloud Platform**: Cloud hosting and infrastructure.
- **Docker & CI/CD**: Containerization and automated deployment pipelines.
- **Git**: Version control and collaborative development.

## Challenges and Learnings

Designing a complaints system for a large retail corporation required careful modeling of the workflow states, escalation rules and reporting needs. Coordinating with Corporación Favorita's internal teams to understand their processes and translate them into a clean data model was a key part of the project.

## Outcome

The portal centralized and standardized the complaints management process across Corporación Favorita's operations, replacing manual or fragmented processes with a single tracked workflow. This improved response times and gave management visibility into complaint trends across departments.
